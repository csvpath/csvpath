
## yes()

Yes

yes() always evaluates to True. It is similar to no() and none().

| Purpose    | Value                             |
|:-----------|:----------------------------------|
| Main focus | yes() produces a calculated value |
| Aliases    | yes, true                         |

| Context          | Qualifier   |
|:-----------------|:------------|
| Match qualifiers | onmatch     |
| Value qualifiers | onmatch     |

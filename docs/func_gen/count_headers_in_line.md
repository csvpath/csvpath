
## count_headers_in_line()

count_headers_in_line() returns the number of headers in the current
line.

| Purpose    | Value                                               |
|:-----------|:----------------------------------------------------|
| Main focus | count_headers_in_line() produces a calculated value |

| Context          | Qualifier   |
|:-----------------|:------------|
| Match qualifiers | onmatch     |
| Value qualifiers | onmatch     |

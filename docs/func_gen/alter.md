
Alter

|:------------------------------------------------------------------|
| alter( in: str|''|None, find: str|''|None, replace: str|''|None ) |

| Call signatures                                                                                                                                       |

| Purpose    | Value                               |
|:-----------|:------------------------------------|
| Main focus | alter() produces a calculated value |

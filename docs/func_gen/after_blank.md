
## after_blank()
After blank

Evaluates to True if the immediately preceding physical line was blank
or had no header values.

| Purpose    | Value                                   |
|:-----------|:----------------------------------------|
| Main focus | after_blank() determines if lines match |

| Context          | Qualifier   |
|:-----------------|:------------|
| Match qualifiers | onmatch     |
| Value qualifiers | onmatch     |


No
no() always evaluates to False. It is similar to yes() and none().

| Purpose    | Value                          |
|:-----------|:-------------------------------|
| Main focus | no() determines if lines match |
| Aliases    | no, false                      |

| Context          | Qualifier   |
|:-----------------|:------------|
| Match qualifiers | [36m[3monmatch[0m     |
| Value qualifiers | [36m[3monmatch[0m     |

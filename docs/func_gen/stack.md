
Stack

Returns a stack variable. The stack is created if not found.

| Data signatures          |
|:-------------------------|
| stack( stack name: str ) |

| Call signatures                                              |
|:-------------------------------------------------------------|
| stack( stack name: Variable|Header|Function|Reference|Term ) |

| Purpose    | Value                    |
|:-----------|:-------------------------|
| Main focus | stack() is a side-effect |

| Context          | Qualifier   |
|:-----------------|:------------|
| Match qualifiers | onmatch     |
| Value qualifiers | onmatch     |

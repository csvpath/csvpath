
## blank()

Blank

A line() schema type representing an incompletely known header.

Blank cannot be used outside a line()

| Purpose    | Value                             |
|:-----------|:----------------------------------|
| Main focus | blank() determines if lines match |
| Type       | Blank is a line() schema type     |
| Aliases    | blank, nonspecific, unspecified   |

| Context          | Qualifier         |
|:-----------------|:------------------|
| Match qualifiers | onmatch, distinct |
| Value qualifiers | onmatch           |

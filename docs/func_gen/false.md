
False

no() always evaluates to False. It is similar to yes() and none().

| Purpose    | Value                             |
|:-----------|:----------------------------------|
| Main focus | false() determines if lines match |
| Aliases    | no, false                         |

| Context          | Qualifier   |
|:-----------------|:------------|
| Match qualifiers | onmatch     |
| Value qualifiers | onmatch     |

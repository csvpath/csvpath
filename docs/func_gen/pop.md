
## pop()

Removes and returns the last value added to a stack variable. The
stack is created if not found.

| Data signatures               |
|:------------------------------|
| pop( stack name: None ǁ str ) |

| Call signatures                                                    |
|:-------------------------------------------------------------------|
| pop( stack name: Variable ǁ Header ǁ Function ǁ Reference ǁ Term ) |

| Purpose    | Value                             |
|:-----------|:----------------------------------|
| Main focus | pop() produces a calculated value |

| Context          | Qualifier       |
|:-----------------|:----------------|
| Match qualifiers | onmatch, asbool |
| Value qualifiers | onmatch         |

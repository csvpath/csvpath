
Divide

Divides numbers. divide() can take any number of int and/or float
arguments.

| Data signatures                                        |
|:-------------------------------------------------------|
| divide( dividend: int|float, divisor: int|float, ... ) |

| Call signatures                                                                                                    |
|:-------------------------------------------------------------------------------------------------------------------|
| divide( dividend: Term|Variable|Header|Function|Reference, divisor: Term|Variable|Header|Function|Reference, ... ) |

| Purpose    | Value                                |
|:-----------|:-------------------------------------|
| Main focus | divide() produces a calculated value |

| Context          | Qualifier   |
|:-----------------|:------------|
| Match qualifiers | onmatch     |
| Value qualifiers | onmatch     |

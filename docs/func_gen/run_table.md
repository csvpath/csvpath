
## run_table()

Prints a table with all the metadata names and values available at
each line. The table is text formatted.

| Purpose    | Value                        |
|:-----------|:-----------------------------|
| Main focus | run_table() is a side-effect |

| Context          | Qualifier   |
|:-----------------|:------------|
| Match qualifiers | onmatch     |
| Value qualifiers | onmatch     |
